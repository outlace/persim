Setup
-------

Installation
~~~~~~~~~~~~

Install persim with pip:

::

    pip install persim

To install from source:

::

    git clone https://github.com/sauln/persim
    cd persim
    pip install -e .


Dependencies
~~~~~~~~~~~~

persim requires:

- Python (>= 2.7 or >= 3.3)
- Scipy
- NumPy
- Scikit-learn
